# Code of Conduct

## Our standards

This project welcomes contributors of all experience levels and backgrounds. Expected behaviour:

- Be respectful and constructive in discussions and reviews
- Focus feedback on code and ideas, not people
- Accept that maintainers have final say on project direction

## Unacceptable behaviour

- Harassment, personal attacks, or discriminatory language
- Deliberate disruption of discussions

## AI-assisted development

NF0T-Logger is developed with AI assistance — specifically [Claude](https://claude.ai) by Anthropic — and we are openly welcoming of AI-assisted contributions. There is no stigma here around using AI tools to write, review, or refactor code.

The expectation is the same regardless of how code was produced: contributors should understand what they are submitting, be able to discuss it in a review, and stand behind its correctness. AI-generated code that the contributor cannot explain or defend is held to the same standard as any other contribution.

## Enforcement

Violations can be reported to **rbutler@tsss.org**. Maintainers reserve the right to remove comments or ban contributors who repeatedly violate these standards.
